## 任务
### 1 简单数据处理与可视化
我们将使用Python进行一个简单的数据处理与可视化. 你需要读入一个`csv`格式文件, 其格式如下:
```csv
index, X, Y
1, 1.4, 6.1
2, 5.12, 8.13
3, -1.4, -7.4
...
```
含有index, X, Y三列. 你需要计算每一行对应的X-Y的值, 然后平方并取对数. 即:

$$ Z = \ln{(X - Y)^2} $$

最后使用`matplotlib`库绘制折线图, 横轴为index, 纵轴为Z.
